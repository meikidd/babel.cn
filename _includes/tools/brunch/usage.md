That's it! Set babel options in your brunch config (such as `brunch-config.coffee`) except
for `filename` and `sourceMap` which are handled internally.

```coffee
plugins:
  ESbabel:
    whitelist: ["arrowFunctions"]
    format:
      semicolons: false
```
